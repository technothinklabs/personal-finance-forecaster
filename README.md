# 💰 Personal Finance Forecast & Tracker

An end-to-end Data Science project built with **Scikit-Learn** and **Streamlit**.

## 🚀 Features
- **Forecasting**: Predicts next day/week/month spending using Random Forest Regression.
- **Visualizations**: Interactive spending trends and category breakdowns via Plotly.
- **ML Pipeline**: Automated scaling and feature engineering.

## 🛠️ Tech Stack
- Python (Scikit-Learn, Pandas, Numpy)
- Streamlit (UI)
- Plotly (Visuals)
- Docker (Containerization)

## 📦 How to Run
1. `pip install -e .`
2. `streamlit run app.py`
